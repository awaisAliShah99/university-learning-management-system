# university-learning-management-system
University LMS Website Project

Project Overview
This project is a university Learning Management System website. The system is designed to manage academic activities for students and faculty in an online environment.

Purpose
The purpose of this project is to provide a centralized platform where students can access learning materials and faculty can manage courses and assignments efficiently.

Main Features
Students can register and log in to the system.
Students can view course materials and submit assignments.
Faculty can upload lectures and assignments.
Faculty can review submissions and provide feedback.

Technologies Used
Frontend technologies include HTML, CSS, and JavaScript.
The project was developed using VS Code.
Git and GitHub are used for version control.

How to Run
Open the project folder in a browser.
Start from the main index page.
Ensure all files remain in the same directory structure.

Conclusion
This LMS website simplifies university academic management by providing an organized and digital learning platform.
